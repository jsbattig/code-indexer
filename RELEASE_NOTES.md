### Using pip
```bash

```

### From Source
```bash

```

## Getting Started

1. **Initialize and Start Services**
   ```bash
   code-indexer init
   code-indexer start
   ```

2. **Index Your Code**
   ```bash
   code-indexer index /path/to/your/project
   ```

3. **Search Your Code**
   ```bash
   code-indexer search "your search query"
   ```

For detailed usage instructions, see the [README](README.md).
